How to use: TurtleController

Welcome to my project! If you're here, you're probably wondering what this is and how it works. This is a project of mine that allows you to control robots from a minecraft mod called ComputerCraft from outside of the game. In this page, I will explain how to effectively use this project.
Please note that anything in red is marked as unfinished, and will be implemented soon. Additionally, anytime this website mentions a turtle, it is referring to a robot.

When the webpage is loaded up, it should prompt you for a link to the server it will connect to. This link is not the link to the minecraft server, it is a different server entierly. The way this project works is that there is server software availible on the Github repository that you can download, run with node.js, and connect to. It will give you the correct server link in the console. If, for whatever reason, the window that this software/terminal is running in closes, the server will shutdown.

Moving the robot

In order to move the robot, you can utilize the 6 buttons in the bottom - left corner. These buttons will move the selected robot up, down, forward, and backward, as well as turning it left and right. There are also keyboard commands that can be found here. The world that this program will display will be populated by the blocks that the robot discovers. For example, if I move my robot over a block of stone, that block of stone will show up in the display on the website. The color of the blocks made will be based on the block id, meaning that all stone will be the same color, but may not be the gray texture we all know and love. The robot will be able to see the block in front of it, above it, and below it. You can hover over the cubes with your mouse to see what block they represent.

Moving the viewpoint

The viewpoint, or camera, or bit-that-shows-you-what-you-are-looking-at, or whatever you want to call it, can be moved by clicking and dragging on anywhere that isn't the control panels. You can also change the level of zoom (how far you are zoomed in) with the scroll wheel.

Fuel counter and robot switcher

The fuel counter and robot switcher will show up in the top left corner. The fuel display will fill in based on how much fuel the robot has, versus the fuel limit it has. It will also change color based on specific thresholds, but you're probably ok if you are in the yellow for fuel. The robot switcher is a set of buttons made to swap control of the robots connected to the aformentioned server, with a few specfied numbers related to that information. They should show up in order of connection, and the id number will always start at 0, whereas the number of connected robots always should start at 1. When switching robots, the camera will aslo move with you, bringing with it the same offset that you were viewing the old robot with, and updating the fuel and inventory display.

Inventory

The inventory display is the 4x4 grid of boxes in the bottom right corner. It will show you the amount of items in the slots of the robot, and, on a right click, will show you the id/name of the item, as well as any actions that you can do with it, such as equipping it, placing it, throwing it, etc. The discolored, lighter square is the currently selected slot. A planned addition to this is an area that tells you what item the robot has equipped, if anything.

Keyboard Controls

This program also accepts some keyboard controls. They are as follows:
Up Arrow and W Moves current robot forward
Down Arrow and S Moves current robot backward
Left Arrow and A Turns current robot left
Right Arrow and D Turns current robot right
> (or period) Changes to next turtle, if possible
< (or comma) Changes to previous turtle, if possible
Space Moves current robot up
Left Shift Moves current robot down
